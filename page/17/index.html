Map

Map

HashMap

  • 存储键值映射关系,不能出现重复的键,可以允许键值为null,

  • 通过put、get存取元素,如果有键形同则覆盖其值,

  • 通过keySet迭代

  • 通过entrySet迭代
    ```java
    Map map = new HashMap<>();

    map.put("1",   "a");
map.put("2", "b");
map.put("3", "c");
map.put("3", "d");

//获得所有键放入set集合中
Set<Entry<String,Object>> entrySet = map.entrySet();
//通过迭代器取出map中的键值关系，迭代器接收的泛型参数和Set接收的一致  
Iterator<Entry<String, Object>> iterator = entrySet.iterator();
while(iterator.hasNext()) {
    //将键值取出放入map.entry接口中
    Map.Entry<String, Object> mapEntry = iterator.next();
    String key = mapEntry.getKey();
    Object value = mapEntry.getValue();

    System.out.println(key+":"+value);

}

输出：
1:a
2:b
3:d

```

list 元素有序可重复

ArrayList

  • 可看做长度可变的数组,
  • 数据结构允许通过索引访问元素，对于数据的查询速度较快,
  • 在增加和删除指定未知的元素时, 会导致创建新的数组效率较低,

    LinkedList

  • 内部维护双向循环链表,对于增删操作有很高的效率

set 元素无序，存入的元素不重复

  • hashset 添加元素时会调用当前存入对象的hashcode方法获得表hash值，根据哈希值确定存储位置，再根据存储位置是否有对象存在进行判断，有对象调用equals方法比较对象是否相等···存入集合

  • TreeSet
    内部结构采用平衡二叉树存储元素,可以保证元素没有重复,还可以对元素进行排序,

多线程

多线程

  • 线程状态
  • 新建、就绪、阻塞、运行、死亡
  • 阻塞状态：调用join()方法或者sleep()方法,join()中断sleep()完成就会回到就绪状态、
  • 调用wait()方法、线程处于等待状态,notify()或者notifyAll()方法唤醒,进入就绪状态、
  • yield()方法：线程让步、不会使线程进入阻塞状态、转换到就绪状态,让调度器重新调度一次、
  • Thread 相关方法
