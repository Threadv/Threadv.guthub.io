String

string&stringbuffer&stringbuilder区别

string

  • 字符串常量，字符串长度不可变。

    stringBuffer

  • 字符串变量（synchronized,线程安全的）如果要频繁对字符串内容进行修改，出于效率考虑最好使用StringBuffer，如果想转成String类型，可以调用StringBuffer的toString()方法。

    StringBuilder

  • 字符串变量（非线程安全）。
