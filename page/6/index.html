Exceutor

Executor

Java通过Executors提供四种线程池，分别为：

  • newCachedThreadPool创建一个可缓存线程池，如果线程池长度超过处理需要，可灵活回收空闲线程，若无可回收，则新建线程。
  • newFixedThreadPool 创建一个定长线程池，可控制线程最大并发数，超出的线程会在队列中等待。
  • newScheduledThreadPool 创建一个定长线程池，支持定时及周期性任务执行。
  • newSingleThreadExecutor 创建一个单线程化的线程池，它只会用唯一的工作线程来执行任务，保证所有任务按照指定顺序(FIFO, LIFO, 优先级)执行。

手动添加修改信息

手动添加修改信息

```php

 public function edit()
{
    $dbConfig = config("db_fenxiao_config");
    $request = Request::instance();
    if ($request->isPost()) {
        $addressData = input('post.');
        if (isset($addressData['id'])) {
            $expressModel = Db::connect($dbConfig)->table("fx_express");
            $result = $expressModel->update($addressData);
            if ($result !== false) {
                $this->success('恭喜，保存成功哦！', '');
            } else {
                $this->error('保存失败，请稍候再试！');
            }
        }
    } elseif ($request->isGet()) {
        $id = $_GET['id'];

        $expressModel = Db::connect($dbConfig)->table("fx_express");
        $vo = $expressModel->where('id', $id)->find();

        if (isset($vo)) {
            $this->assign('vo', $vo);
        }
        $this->assign('id', $id);
        return $this->fetch('form');
    } else {
        return null;
    }
    return null;


}
