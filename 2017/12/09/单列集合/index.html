单列集合

list 元素有序可重复

ArrayList

  • 可看做长度可变的数组,
  • 数据结构允许通过索引访问元素，对于数据的查询速度较快,
  • 在增加和删除指定未知的元素时, 会导致创建新的数组效率较低,

    LinkedList

  • 内部维护双向循环链表,对于增删操作有很高的效率

set 元素无序，存入的元素不重复

  • hashset 添加元素时会调用当前存入对象的hashcode方法获得表hash值，根据哈希值确定存储位置，再根据存储位置是否有对象存在进行判断，有对象调用equals方法比较对象是否相等···存入集合

  • TreeSet
    内部结构采用平衡二叉树存储元素,可以保证元素没有重复,还可以对元素进行排序,
